<objective>
## Goal
Build the FxChain DSP class: multimode LP filter, reverb, and feedback delay with per-param
SmoothedValue smoothing (20ms ramp) for click-free p-lock transitions. Wire into PluginProcessor
so FxParams (snapshotted in 06-01) drives actual audio processing.

## Purpose
06-01 established the p-lock pipeline ending with `fx_params` ready in processBlock() but
marked `// fx_params pronto para FxChain::process() (Fase 6-02)` — this plan delivers that call.
After 06-02 every p-lock parameter change has audible real-time effect. Sidechain compressor is
deferred to 06-03 (Phase 6 DoD: "sidechain pump functional").

## Architecture
```
processBlock()
    │ fx_params (from APVTS snapshot + PLockBatch overrides — 06-01)
    ▼
FxChain::process(buffer, fx_params)
    ├─ LP filter (StateVariableTPTFilter, SmoothedValue for cutoff + resonance)
    ├─ Reverb     (juce::dsp::Reverb, SmoothedValue for mix — wet/dry blend)
    └─ Delay      (juce::dsp::DelayLine stereo, SmoothedValue for mix + time)
```

Key invariant: all processing in FxChain is RT-safe — no allocation, no locks.
SmoothedValue::skip(numSamples) advances per block; coefficients updated once per block.

## Task 1 — Create `src/audio/fx_chain.h` and `src/audio/fx_chain.cpp`

### fx_chain.h

```cpp
#pragma once
#include "fx_params.h"

#include <juce_audio_basics/juce_audio_basics.h>
#include <juce_dsp/juce_dsp.h>

// Cadeia de FX de áudio do BAQUE — filtro, reverb, delay com suavização por SmoothedValue.
// RT-safe: sem alocação, sem locks em process(). prepare() aloca; chamar off audio thread.
// sidechain_threshold em FxParams ignorado aqui — roteado na Fase 6-03.
class FxChain {
  public:
    // Aloca buffers internos e configura DSP. Chamar em prepareToPlay(), nunca em processBlock().
    void prepare(double sample_rate, int max_block_size) noexcept;

    // Aplica cadeia de FX ao buffer estéreo. fx_params define alvos dos SmoothedValues.
    // Chamado em processBlock() após renderização de vozes + aplicação de ganho.
    void process(juce::AudioBuffer<float>& buffer, const FxParams& params) noexcept;

    // Zera estado interno (delay buffer, filter state, reverb tail). Chamar em releaseResources().
    void reset() noexcept;

  private:
    double sample_rate_ = 44100.0;
    bool is_prepared_ = false;

    // Filtro passa-baixas (StateVariableTPT — um por canal para processamento per-sample)
    juce::dsp::StateVariableTPTFilter<float> filter_l_;
    juce::dsp::StateVariableTPTFilter<float> filter_r_;
    juce::SmoothedValue<float> cutoff_smoother_;
    juce::SmoothedValue<float> res_smoother_;

    // Reverb (juce::dsp::Reverb — estéreo nativo)
    juce::dsp::Reverb reverb_;
    juce::SmoothedValue<float> reverb_mix_smoother_;

    // Delay estéreo com interpolação linear
    juce::dsp::DelayLine<float, juce::dsp::DelayLineInterpolationTypes::Linear> delay_line_{192000};
    juce::SmoothedValue<float> delay_mix_smoother_;
    juce::SmoothedValue<float> delay_time_smoother_;

    static constexpr float k_smooth_time_s = 0.02f;  // 20ms ramp — sem cliques em p-lock
    static constexpr float k_delay_feedback = 0.45f; // feedback 45% — sem self-oscillation
    static constexpr float k_reverb_room    = 0.6f;
    static constexpr float k_reverb_damp    = 0.5f;
    static constexpr float k_reverb_width   = 1.0f;
};
```

### fx_chain.cpp

Implement `prepare()`, `process()`, `reset()`. Key behaviors:

**prepare():**
- Set `sample_rate_`; set `is_prepared_ = true`
- `juce::dsp::ProcessSpec spec{sample_rate, (uint32_t)max_block_size, 2}` — stereo
- Filter: prepare each with mono spec `{sample_rate, (uint32_t)max_block_size, 1}`; set type to `juce::dsp::StateVariableTPTFilterType::lowpass`
- Reverb: `reverb_.prepare(spec)`
- Delay: `delay_line_.prepare(spec)` (spec.numChannels=2 for stereo DelayLine)
- SmoothedValues: `smoother.reset(sample_rate, k_smooth_time_s)` for each; set initial values:
  - cutoff: 20000.0f, res: 0.707f (Q for Butterworth), reverb_mix: 0.0f, delay_mix: 0.0f, delay_time: 0.25f
  - Use `setCurrentAndTargetValue()` so first block has no ramp artifact

**process():**
- `jassert(is_prepared_)` at top; return if not prepared (no UB)
- Set targets: `cutoff_smoother_.setTargetValue(params.filter_cutoff)`, similarly for all 5 params
- Advance smoothers via `skip(numSamples)` to get end-of-block values (block-rate coefficient update):
  ```cpp
  const float cutoff  = juce::jlimit(20.0f, 20000.0f, cutoff_smoother_.skip(num_samples));
  const float res_q   = juce::jlimit(0.1f,  20.0f, res_smoother_.skip(num_samples) * 20.0f + 0.1f);
  const float rev_mix = juce::jlimit(0.0f,  1.0f,  reverb_mix_smoother_.skip(num_samples));
  const float dly_mix = juce::jlimit(0.0f,  1.0f,  delay_mix_smoother_.skip(num_samples));
  const float dly_t   = juce::jlimit(0.001f, 2.0f, delay_time_smoother_.skip(num_samples));
  ```
  Note: `FxParams::filter_res` is [0,1]; map to Q [0.1, 20.1] for the filter.
- **Filter:** update both filter_l_ and filter_r_ coefficients, then process per sample:
  ```cpp
  filter_l_.setCutoffFrequency(cutoff); filter_r_.setCutoffFrequency(cutoff);
  filter_l_.setResonance(res_q);        filter_r_.setResonance(res_q);
  for (int i = 0; i < num_samples; ++i) {
      left[i]  = filter_l_.processSample(0, left[i]);
      right[i] = filter_r_.processSample(0, right[i]);
  }
  ```
  Use `buffer.getWritePointer(0)` for left; if mono (numChannels < 2), use left for both.

- **Reverb:** set params with wetLevel=rev_mix, dryLevel=1.0f-rev_mix, roomSize/damping/width from constants:
  ```cpp
  // KNOWN PATTERN (audit SR1): setParameters() called every block even when params unchanged.
  // RT-safe: no allocation, noexcept — recalculates Comb/AllPass coefficients in-place.
  // Dirty-flag optimization deferred to Phase 12 (Hardening).
  juce::dsp::Reverb::Parameters rp{};
  rp.roomSize = k_reverb_room; rp.damping = k_reverb_damp;
  rp.wetLevel = rev_mix;       rp.dryLevel = 1.0f - rev_mix;
  rp.width    = k_reverb_width; rp.freezeMode = 0.0f;
  reverb_.setParameters(rp);
  juce::dsp::AudioBlock<float> block(buffer);
  reverb_.process(juce::dsp::ProcessContextReplacing<float>(block));
  ```

- **Delay:** per-sample push/pop on stereo DelayLine (channel 0=L, 1=R):
  ```cpp
  const int delay_samples = juce::roundToInt(dly_t * static_cast<float>(sample_rate_));
  delay_line_.setDelay(static_cast<float>(delay_samples));
  // INVARIANT (audit SR2): pop BEFORE push — read delayed sample first, then write current
  // sample to the delay buffer. Reversing order introduces 1-sample feedback error.
  for (int i = 0; i < num_samples; ++i) {
      const float wet_l = delay_line_.popSample(0);
      const float wet_r = (num_channels > 1) ? delay_line_.popSample(1) : wet_l;
      delay_line_.pushSample(0, left[i]  + wet_l * k_delay_feedback);
      if (num_channels > 1)
          delay_line_.pushSample(1, right[i] + wet_r * k_delay_feedback);
      left[i]  = left[i]  * (1.0f - dly_mix) + wet_l * dly_mix;
      if (num_channels > 1)
          right[i] = right[i] * (1.0f - dly_mix) + wet_r * dly_mix;
  }
  ```

**reset():**
- `filter_l_.reset()`, `filter_r_.reset()`, `reverb_.reset()`, `delay_line_.reset()`
- Reset all SmoothedValues to their current target (no ramp artifacts on re-prepare):
  `cutoff_smoother_.setCurrentAndTargetValue(cutoff_smoother_.getTargetValue())`

```cpp
static FxParams default_params() { return FxParams{}; }

static void fill_sine(juce::AudioBuffer<float>& buf, float freq, double sr) {
    for (int ch = 0; ch < buf.getNumChannels(); ++ch)
        for (int i = 0; i < buf.getNumSamples(); ++i)
            buf.setSample(ch, i, std::sin(2.0 * M_PI * freq * i / sr));
}

static float peak_amplitude(const juce::AudioBuffer<float>& buf) {
    float peak = 0.0f;
    for (int ch = 0; ch < buf.getNumChannels(); ++ch)
        for (int i = 0; i < buf.getNumSamples(); ++i)
            peak = std::max(peak, std::abs(buf.getSample(ch, i)));
    return peak;
}
```

**FC5 — delay_mix=0 dry; delay stores/replays signal across blocks:**
```cpp
TEST_CASE("FC5 - delay mix 0 is dry-only; positive mix replays signal", "[fx_chain]") {
    juce::ScopedJuceInitialiser_GUI init;
    {
        FxChain fx;
        fx.prepare(44100.0, 512);
        FxParams p{}; p.delay_mix = 0.0f; p.delay_time = 0.01f;
        juce::AudioBuffer<float> in_buf(2, 512), out_buf(2, 512);
        fill_sine(in_buf, 440.0f, 44100.0);
        out_buf.copyFrom(0, 0, in_buf, 0, 0, 512);
        out_buf.copyFrom(1, 0, in_buf, 1, 0, 512);
        fx.process(out_buf, p);
        // With delay_mix=0, output should equal input
        for (int i = 0; i < 512; ++i)
            REQUIRE(out_buf.getSample(0, i) == Catch::Approx(in_buf.getSample(0, i)).epsilon(1e-5));
    }
    {
        FxChain fx;
        fx.prepare(44100.0, 512);
        FxParams p{}; p.delay_mix = 0.5f; p.delay_time = 0.001f; // 44 samples delay
        juce::AudioBuffer<float> buf(2, 512);
        fill_sine(buf, 440.0f, 44100.0);
        fx.process(buf, p); // first block — fills delay buffer
        fill_sine(buf, 440.0f, 44100.0);
        const float before = buf.getSample(0, 100);
        fx.process(buf, p); // second block — delay contribution present
        const float after = buf.getSample(0, 100);
        REQUIRE(std::abs(after - before) > 1e-4f); // delay changed the output
    }
}
```
