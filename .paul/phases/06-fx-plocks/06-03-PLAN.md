<objective>
## Goal
Build the SidechainCompressor DSP class: asymmetric IIR envelope follower + hard-knee gain
computer (8:1 ratio, 5ms attack, 200ms release). Wire into FxChain as the last processing unit,
driven by FxParams::sidechain_threshold. Satisfies Phase 6 DoD item: "sidechain pump functional".

## Purpose
06-01 defined `sidechain_threshold` in FxParams and the APVTS; 06-02 built FxChain but explicitly
deferred sidechain DSP (comment: "sidechain_threshold em FxParams ignorado — roteado na Fase 6-03").
After this plan, a p-lock on sidechain_threshold produces an audible pump effect — loud transient
charges the envelope, compression kicks in, 200ms release creates the swell characteristic of
groove music (Dilla, UK Garage, deep house pumping bass).

## Architecture
```
FxChain::process()
    ├─ LP filter          (06-02 — per-sample SVTPT)
    ├─ Reverb             (06-02 — juce::dsp::Reverb)
    ├─ Delay              (06-02 — stereo DelayLine + feedback)
    └─ SidechainCompressor (06-03) ← LAST, compresses full mixed signal
         key signal: mono peak of stereo buffer (internal trigger)
         threshold: FxParams::sidechain_threshold (dBFS, p-lockable)
         ratio/attack/release: fixed constants (exposed in Phase 10 UI if needed)
```

Placement LAST: reverb tail and delay echoes are compressed together with the dry signal.
This is the standard "ducking reverb" technique — reverb disappears on the beat, swells back.

## Task 1 — Create `src/audio/sidechain_comp.h` and `src/audio/sidechain_comp.cpp`

### sidechain_comp.h

```cpp
#pragma once
#include <juce_audio_basics/juce_audio_basics.h>

// Compressor de sidechain com trigger interno — envelope follower assimétrico (IIR) + gain computer.
// RT-safe: sem alocação em process(). Coeficientes pré-computados em prepare().
// threshold via FxParams::sidechain_threshold (dBFS). Ratio/attack/release são constantes nesta fase
// (Fase 10 expõe no UI se necessário). Posicionado ÚLTIMO na FxChain (pós-reverb/delay).
class SidechainCompressor {
  public:
    void prepare(double sample_rate, int max_block_size) noexcept;
    void process(juce::AudioBuffer<float>& buffer, float threshold_db) noexcept;
    void reset() noexcept;

  private:
    double sample_rate_ = 44100.0;
    bool is_prepared_ = false;
    float env_ = 0.0f;           // envelope follower state (linear amplitude, ≥0)
    float attack_coeff_ = 0.0f;  // IIR coeff for attack (key > env_)
    float release_coeff_ = 0.0f; // IIR coeff for release (key < env_)

    static constexpr float k_ratio = 8.0f;        // 8:1 — pump agressivo
    static constexpr float k_attack_s = 0.005f;   // 5ms — detecta transiente rápido
    static constexpr float k_release_s = 0.200f;  // 200ms — release lento → swell característico
};
```

### sidechain_comp.cpp

**prepare():**
```cpp
void SidechainCompressor::prepare(double sample_rate, int /*max_block_size*/) noexcept {
    sample_rate_ = sample_rate;
    env_ = 0.0f;
    const float sr = static_cast<float>(sample_rate);
    attack_coeff_  = std::exp(-1.0f / (k_attack_s  * sr));
    release_coeff_ = std::exp(-1.0f / (k_release_s * sr));
    is_prepared_ = true;
}
```

**process():**
```cpp
void SidechainCompressor::process(juce::AudioBuffer<float>& buffer, float threshold_db) noexcept {
    jassert(is_prepared_);
    if (!is_prepared_) return;
    const int num_samples = buffer.getNumSamples();
    const int num_channels = buffer.getNumChannels();
    if (num_samples == 0 || num_channels == 0) return;

    auto* left  = buffer.getWritePointer(0);
    auto* right = (num_channels > 1) ? buffer.getWritePointer(1) : nullptr;
    // Clamp before conversion — NaN/extremos de APVTS causariam buffer de NaN (SR2, audit 06-03)
    const float threshold_linear = juce::Decibels::decibelsToGain(
        juce::jlimit(-80.0f, 0.0f, threshold_db));

    for (int i = 0; i < num_samples; ++i) {
        // Key signal: mono peak (internal trigger from own output)
        float key = std::abs(left[i]);
        if (right) key = std::max(key, std::abs(right[i]));

        // Asymmetric IIR envelope follower
        const float coeff = (key > env_) ? attack_coeff_ : release_coeff_;
        env_ = coeff * env_ + (1.0f - coeff) * key;

        // Gain computer: hard knee at threshold
        float gain = 1.0f;
        if (env_ > threshold_linear) {
            const float env_db    = juce::Decibels::gainToDecibels(env_);
            const float excess_db = env_db - threshold_db;            // > 0
            const float gain_db   = excess_db * (1.0f / k_ratio - 1.0f); // < 0
            gain = juce::Decibels::decibelsToGain(gain_db);           // < 1.0
        }

        left[i] *= gain;
        if (right) right[i] *= gain;
    }
}
```

**reset():**
```cpp
void SidechainCompressor::reset() noexcept {
    env_ = 0.0f;
}
```

**Includes needed in .cpp:** `"sidechain_comp.h"`, `<cmath>`, `<algorithm>` (for `std::max`, `std::abs`).

**Do not:** add SmoothedValue (IIR envelope IS the temporal smoothing); do not compute attack/release coefficients per-sample (pre-computed in prepare()); do not use `juce::dsp::Compressor` (need custom envelope for the pump characteristic).

### tests/test_sidechain_comp.cpp

```cpp
#include "audio/sidechain_comp.h"
#include <juce_audio_processors/juce_audio_processors.h>
#include <catch2/catch_approx.hpp>
#include <catch2/catch_test_macros.hpp>
#include <cmath>

static constexpr double k_pi = 3.14159265358979323846;

static void fill_sine(juce::AudioBuffer<float>& buf, float freq, double sr, float amplitude = 1.0f) {
    for (int ch = 0; ch < buf.getNumChannels(); ++ch)
        for (int i = 0; i < buf.getNumSamples(); ++i)
            buf.setSample(ch, i, amplitude * static_cast<float>(std::sin(2.0 * k_pi * freq * i / sr)));
}

static float peak_amplitude(const juce::AudioBuffer<float>& buf) {
    float peak = 0.0f;
    for (int ch = 0; ch < buf.getNumChannels(); ++ch)
        for (int i = 0; i < buf.getNumSamples(); ++i)
            peak = std::max(peak, std::abs(buf.getSample(ch, i)));
    return peak;
}
```

**SC5 — threshold controls compression (contrast test):**
```cpp
TEST_CASE("SC5 - threshold controls compression: low threshold compresses, 0 dBFS transparent", "[sidechain]") {
    juce::ScopedJuceInitialiser_GUI init;
    // Contrast: amplitude=0.5 (-6 dBFS) is ABOVE -12 dBFS threshold but BELOW 0 dBFS threshold.
    // audit SR1: original single-case test was vacuous (IIR env_ ≤ 1.0 for unit-amplitude signal,
    // so threshold_linear=1.0 branch never entered). This contrast version cannot vacuously pass.
    auto measure_tail = [](float threshold_db) -> float {
        SidechainCompressor comp;
        comp.prepare(44100.0, 4096);
        juce::AudioBuffer<float> buf(2, 4096);
        fill_sine(buf, 440.0f, 44100.0, 0.5f); // -6 dBFS: above -12dBFS threshold, below 0dBFS
        comp.process(buf, threshold_db);
        float peak = 0.0f;
        for (int ch = 0; ch < 2; ++ch)
            for (int i = 512; i < 4096; ++i) // skip attack transient
                peak = std::max(peak, std::abs(buf.getSample(ch, i)));
        return peak;
    };
    // threshold=-12 dBFS: 0.5 amplitude is 6 dB above → compression engaged
    // envelope→0.5; env_db=-6; excess=6; gain_db=6*(-0.875)=-5.25dB; gain≈0.546; out≈0.273
    REQUIRE(measure_tail(-12.0f) < 0.5f * 0.7f); // >30% reduction (contrast case proves threshold works)
    // threshold=0 dBFS: threshold_linear=1.0; 0.5 amplitude → envelope→0.5 < 1.0 → no compression
    REQUIRE(measure_tail(0.0f) > 0.5f * 0.9f);   // <10% reduction (proves threshold=0 disables)
}
```
