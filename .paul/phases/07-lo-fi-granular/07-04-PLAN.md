```gherkin
Given FxChain filled with 4096 samples of 440Hz sine (freeze=false via FxChain)

     (grains replay frozen capture buffer; LP/reverb/delay default params pass it through)
```

```gherkin
     AND instance B processes 512 samples with granular_spray=0.5f

```
